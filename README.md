# Am I a Commie? Web App

This is a static, single-page web app. It needs no server, database, or build step.

## Files
- `index.html` — the quiz app
- `manifest.json` — PWA install metadata
- `sw.js` — offline cache service worker
- `icon-192.png` and `icon-512.png` — app icons

## Free GitHub Pages hosting
1. Create a new GitHub repository, for example `am-i-a-commie`.
2. Upload every file in this folder to the repository root.
3. In GitHub, go to **Settings → Pages**.
4. Under **Build and deployment**, choose **Deploy from a branch**.
5. Select branch `main` and folder `/root`, then save.
6. GitHub will provide a live link like `https://YOURUSERNAME.github.io/am-i-a-commie/`.

Once hosted, the Share Results button and QR code point to the live quiz URL.
